What Our App Does

            

By displaying information in a digestable dashboard, the Deforestation Dashboard gives users insight  into how the world's forest cover is declining and where it is actually increasing.

Deforestation Dashboard shows how changes in forest cover vary by country/region over time,  and can forecast future forest cover changes by using time-series data.

This allows for greater understanding about forest loss and its effects,  which can provide sharper focus for conservation efforts.
